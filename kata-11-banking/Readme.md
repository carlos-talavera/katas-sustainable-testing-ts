# Kata 11 - Banking

In this kata, a basic banking is implemented using a TDD approach and [Jest](https://jestjs.io).

## Compile the project

To compile the project, run:

```shell
npm run compile
```

## Watch for changes

To make changes and automatically update the build, run:

```shell
npm run compile:watch
```

## Run tests

Run the tests:

```shell
npm run test
```
